# Репозиторий мини-проекта "Сопоставление признаков на изображениях"

## Структура репозитория:
- Директория `src` содержит исходный код проекта, в котором представлена реализация 
  детекторов, дескрипторов и сопоставителей, а также модули для вычисления метрик
  качества и производительности.
- Директория `samples` содержит тестовые приложения для решения различных задач.
- Директория `tests` содержит тесты корректной работоспособности различных модулей проекта.
- Директория `results` содержит результаты измерения метрик качества и производительности
  решения задачи сопоставления признаков.
